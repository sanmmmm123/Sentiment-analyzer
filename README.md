# Sentiment-analyzer
My first AI/NLP project built using Python and Machine Learning.

## What it does
Takes a text sentence and classifies it into a category using Machine Learning.

## Technologies Used
- Python
- scikit-learn
- NLTK
- Pandas
- Google Colab

## How it works
1. Load text dataset
2. Clean the text (remove stopwords, symbols)
3. Convert text to numbers using TF-IDF
4. Train Logistic Regression model
5. Predict category of new sentences

## Results
- Model Accuracy: 96.78%

## What I learned
- Data cleaning
- NLP basics
- TF-IDF Vectorization
- Machine Learning with scikit-learn
